Escape Room documentation

Introduction

Escape room is theme, which is created to make impression on you and your customers. We offer you an up-to-date and really high quality work with outstanding design and well-thought usability approach. This template can help you to achieve a significant increase of the conversion rates and number of positive feedback, as well as will simply be a nice surprise for your customers.

Project structure

Template - style files, scripts, html elements and pages are located here.

css – standard style libraries used in the template.

  • animate.css - https://daneden.github.io/animate.css/
  • bootstrap.min.css - http://getbootstrap.com/customize/
  • font-awesome.min.css - http://fontawesome.io/
  • pe-icon-7-stroke.css - http://themes-pixeden.com/font-demos/7-stroke/
  • slick.css & slick-theme.css - http://kenwheeler.github.io/slick/
  • style.css & responsive.css – main css files.

fonts – set of fonts to display the icons used in the template.

  • fontawesome-webfont.* - http://fontawesome.io/
  • pe-icon-7-stroke.* - http://themes-pixeden.com/font-demos/7-stroke/
  • slick.* - http://kenwheeler.github.io/slick/

img – images of template and content images.

video – videos of template.

js – files of scripts used in this template.

  • jquery-1.12.0.min.js - https://jquery.com/
  • jquery.viewportchecker.min.js - https://github.com/dirkgroenen/jQuery-viewport-checker
  • slick.min.js - http://kenwheeler.github.io/slick/
  • map.js - https://mapbuildr.com/
  • animation.js - js file with animation effects for template, based on jquery.viewportchecker.min.js
  • counter.js – js file for animated counter
  • script.js - main js file.
